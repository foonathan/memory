Detailed Description

template<typename T, class RawAllocator, class Mutex = default_mutex>
class foonathan::memory::std_allocator< T, RawAllocator, Mutex >

Wraps a RawAllocator and makes it a "normal" Allocator.

It allows using a RawAllocator anywhere a Allocator is required. It stores the allocator as allocator_reference to allow required copying of a Allocator, synchronization is done via a Mutex which defaults to default_mutex.

Effects:\n Creates it from a reference to a RawAllocator.

Requires:\n The expression allocator_reference<RawAllocator, Mutex>(alloc) is well-formed,
