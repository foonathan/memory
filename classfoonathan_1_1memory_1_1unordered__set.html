Detailed Description

template<typename T, class RawAllocator, class Mutex = default_mutex>
class foonathan::memory::unordered_set< T, RawAllocator, Mutex >

Alias template for an STL container that uses a certain RawAllocator.

It is just a shorthand for a passing in the RawAllocator wrapped in a foonathan::memory::std_allocator. The Mutex type defaults to the foonathan::memory::default_mutex.
